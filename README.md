# Leadzu
Faster work hours inputer

## How to build
1. Get go: https://golang.org/doc/install
2. Use: **go build**

## Config.json
Input your user and password on config.json

````
{
  "user": "",
  "password": "",
  "startHour": "08:00",
  "endHour": "15:00",
  "realHours": "7"
}
````

## Launch
Use: **./leadzu**
